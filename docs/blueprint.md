# **App Name**: Melodif

## Core Features:

- Music Streaming: Play music from various sources.
- AI-Powered Recommendations: Provide personalized music recommendations based on listening history using an AI tool.
- User Playlists: Allow users to create and manage their own playlists.
- Search Functionality: Enable users to search for songs, artists, and albums.
- Music Upload: Allow musicians to upload music and get it hosted in this app.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to evoke a sense of creativity and sophistication.
- Background color: Very light Lavender (#EDE7F6), a desaturated variant of deep indigo to create a calm and focused ambiance.
- Accent color: Violet (#9C27B0) to draw attention to important interactive elements.
- Body and headline font: 'Inter', a sans-serif for a clean, modern look.
- Use modern, line-based icons for a clean and consistent look.
- Clean and intuitive layout for easy navigation and music discovery.
- Subtle transitions and animations for a smooth and engaging user experience.
